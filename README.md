# educa-matinha
Educa Matinha é um projeto educacional desenvolvido pela Clojurehackers em parceria com cuidadores da APP do Riacho São Francisco, proposto como atividade extensionista vinculada à Universidade Estadual de Campinas (Unicamp).
## how to run
```clojure -M -m figwheel.main --build educa-matinha --repl```
